# Data Analytics Project – Hotel Prices Zürich

## Project Description
This project analyses hotel prices from Booking.com using web scraping and open data.

## Steps
1. Data Collection (Web Scraping)
2. Data Preparation (Cleaning, Feature Engineering, Open Data Integration)
3. Database Storage (PostgreSQL)
4. Exploratory Data Analysis (EDA)
5. Modeling (Regression, Clustering)
